문제 요약

1. a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1] 구하기

<br>

---

<br>

- 최종 코드

```cpp
#include <string>
#include <vector>

using namespace std;

int solution(vector<int> a, vector<int> b) {
    int answer = 0;
    
    //a.size() = b.size()
    for(int i = 0; i < a.size(); i++)
        answer += a[i] * b[i];
    
    return answer;
}
```
