### 문제 요약

1. 주어진 수 n(가로), m(세로)만큼의 직사각형을 별로 찍기⭐

<br>

---

<br>

### 최종 코드

```cpp
#include <iostream>

using namespace std;

int main(void) {
    int n;
    int m;
    cin >> n >> m;
    
    for(int i = 0; i < m; i++)
    {
        for(int j = 0; j < n; j++)
        {
            cout << '*';
        }
        cout << endl;
    }
    //⭐⭐⭐⭐⭐
    //⭐⭐⭐⭐⭐
    //⭐⭐⭐⭐⭐
    
    return 0;
}
```
