문제 요약

1. 주어진 수를 3진법으로 표현
2. 3진법으로 표현된 수를 앞뒤 반전시킴
3. 반전된 3진법 수를 10진법으로 다시 표현

<br>

---

<br>

- 최종 코드

```cpp
#include <string>
#include <vector>
#include <algorithm>
#include <cmath>

using namespace std;

int solution(int n) {
    int answer = 0;
    vector<int> temp;
    
    while(n != 0)
    {
        temp.push_back(n % 3);
        n /= 3;
    }
    
    reverse(temp.begin(), temp.end());
    
    for(int i = 0; i < temp.size(); i++)
    {
        answer += temp[i] * pow(3, i);
    }
    
    return answer;
}
```

<br>

---

<br>

- 트러블 슈팅
    - reverse 함수를 사용하기 위한 algorithm 헤더, pow를 사용하기 위한 cmath 헤더. 헤더를 자꾸 빼먹는 실수를 한다
